Keyboard shortcut Action
tmux new-session -s <session_name> Starting a Tmux Session
tmux ls Listing out all sessions
Ctrl-b followed by d To detach from a session
tmux attach-session -t <session_name> To reattach a session
Ctrl-b followed by n Switch to the next window
Ctrl-b followed by p Switch to the previous window
Ctrl-b followed by l Switch to the last window
Ctrl-b followed by a number (0-9) Switch to a specific window by index
Ctrl-b followed by c Create a New Window
Ctrl-b followed by w List Windows
Ctrl-b followed by % Create a Horizontal Pane
Ctrl-b followed by " Create a Vertical Pane
Ctrl-b followed by an arrow key Navigate between Panes
Ctrl-b followed by , Rename Window
Ctrl-b followed by [ to enter scroll mode Scroll in Pane
Press / and then type the text to search Search in Scroll Mode
Ctrl-b followed by x Kill Pane
exit or Ctrl-d (within a pane) Exit Tmux
